Begreppsförklaring

Element:

Ett element i HTML är en del av ett HTML dokument som börjar med en starttagg och avslutas med en sluttagg. Allt ifrån ord till bilder och färger kan tillägas med hjälp av olika element. Några exempel är h1, h2, img src.

HTML:

Står för HyperText Markup Language, är del av den grundläggande standarden för World Wide Web. De flesta hemsidor skrivs i HTML och överförs med HTTP.

ID:

Är ett sätt att döpa ett element i html. Som en class fast ID gör elementet unikt. Detta gör att man bara kan använda ID en gång per HTML. Med hjällp av ID:et kan man ändra saker i elementet i css till exemplel.

Class:

Man kan använda sig av classes i html för att namnge ett element och sedan på ett lätt sätt kunna ändra på detta i en css fil till exempel.

Meta:

Meta taggen skriver man inuti headen och dessa taggar bestämmer, teckenuppsättning, sidbeskrivning, nyckelord och ägaren över dokumentet. Dessa taggar synd inte på sidan utan är information som jobbar i bakgrunden och på hela sidan.

Responsive:

Gör att man innehåll ändras beroende på webbläsares olika storlekar, bilder ska t.ex. inte ta upp hela skärmen om man använder mobilen t.ex. Texten ska också ändras efter olika storlekar.
